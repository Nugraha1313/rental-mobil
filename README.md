# rental-mobil
Aplikasi GUI Rental Mobil Menggunakan Java SWING + MySQL

# Installation
## 1. Create DB
```
cd rental-mobil
```
<b> Go To PHP MyAdmin and import rental_mobil.sql in lib folder
Dont Forget to Import Library (JAR) to the project.
The library is in lib directory. </b>
